Este repositório reúne implementações práticas de algoritmos clássicos, desenvolvidas a partir do estudo do livro “Entendendo Algoritmos – Um guia ilustrado para programadores e outros curiosos”, de Aditya Y. Bhargava.
O objetivo principal é transformar conceitos teóricos em código funcional, permitindo:
Melhor compreensão do funcionamento interno dos algoritmos;
Análise da eficiência por meio da notação Big O;
Desenvolvimento do raciocínio lógico e algorítmico;
Consolidação de boas práticas de programação.
Cada implementação busca priorizar clareza, organização e propósito didático, servindo tanto como material de estudo quanto como registro prático da evolução técnica do desenvolvedor.
